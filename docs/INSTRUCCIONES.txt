========================================
    APP-STOCK - SISTEMA DE INVENTARIO
========================================

BIENVENIDO AL SISTEMA DE GESTION DE INVENTARIO Y VENTAS

========================================
    INSTALACION RAPIDA (3 PASOS)
========================================

PASO 1: INSTALAR XAMPP
----------------------
1. Descargue XAMPP desde: https://www.apachefriends.org/
2. Instale XAMPP (solo MySQL y Apache)
3. NO inicie XAMPP al finalizar

PASO 2: CONFIGURAR BASE DE DATOS
-------------------------------
1. Inicie XAMPP Control Panel
2. Haga clic en "Start" junto a MySQL
3. Ejecute el archivo "configurar.bat"
4. Siga las instrucciones en pantalla

PASO 3: EJECUTAR APLICACION
---------------------------
1. Haga doble clic en "App-Stock.exe"
2. ¡Listo! Ya puede usar el sistema

========================================
    FUNCIONALIDADES INCLUIDAS
========================================

✅ GESTION DE PRODUCTOS
   - Agregar, editar, eliminar productos
   - Control de stock automatico
   - Busqueda por codigo de barras

✅ SISTEMA DE VENTAS
   - Carrito de compras intuitivo
   - Calculo automatico de totales
   - Tickets de venta en PDF
   - Impresion directa

✅ REPORTES PROFESIONALES
   - Exportacion a PDF y Excel
   - Graficos de productos mas vendidos
   - Historial completo de ventas
   - Anulacion de ventas

✅ ARTICULOS VARIOS
   - Venta de productos no registrados
   - Ideal para servicios o productos especiales

========================================
    REQUISITOS DEL SISTEMA
========================================

- Windows 10 o superior
- 4 GB RAM minimo
- 500 MB espacio en disco
- Conexion a internet (solo para descargar XAMPP)

========================================
    SOPORTE TECNICO
========================================

Si tiene problemas:

1. Verifique que MySQL este ejecutandose en XAMPP
2. Ejecute "configurar.bat" nuevamente
3. Reinicie la aplicacion
